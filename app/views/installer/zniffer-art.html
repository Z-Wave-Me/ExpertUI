Z-Wave.Me hardware and software has a built-in Z-Wave sniffer (Zniffer) included. But it has limitations: it can only listen for own network, hence requires inclusion in the network for diagnostics. Also, it can only listen for packets in the air.

The new Z-Wave.Me 7th generation hardware introduces the support of a full feature Zniffer and Advance Radio Tool (A.R.T.) — the must have tool for installers and advanced users allowing them to:

  • Inspect Z-Wave packets
  • Decode and analyze Z-Wave commands
  • Decrypt secure Z-Wave commands (Security S0 and S2)
  • Generate Z-Wave commands
  • Make on-site signal strength probes
  • Gather network statistics
  • Proble the network using multiple Zniffers
  • All the above without doing the inclusion in the network

Zniffer/A.R.T. can help in diagnostics of a network with any Z-Wave controller: Z-Way, Fibaro Home Center, Yubii, Vera, Smart Things, Zipato, HomeSeer and others.

A.R.T. software requires RaZberry 7 or Z-Uno 2 as a hardware. To enable the new feature a new firmware (version 7.26 or upper for RaZberry 7 and 3.0.9 for Z-Uno 2). An additional license are required for this feature. Current capabilities can be inspected on the Controller Info page.

A.R.T. can decode secure packets (network key need to be learned from the primary controller by including into the network or by exporting keys), parse received Z-Wave command, construct and send Z-Wave commands.

It is also possible to attach multiple Zniffers placed in different parts of the house to one A.R.T. tool (using LAN or Wi-Fi) to see the same packet visible from different locations. It is also possible to make probes from each Zniffer to measure signal strength at different locations.
